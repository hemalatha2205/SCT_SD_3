# Sudoku Solver

A polished web app that lets users enter a Sudoku puzzle, validates it, and solves it with a backtracking algorithm.

## Features

- 9x9 interactive Sudoku board
- Input validation for empty boards, invalid values, and duplicate entries
- Solve animation with timer feedback
- Sample puzzles for easy, medium, and hard difficulty
- Clear and theme toggle controls

## Run locally

Open index.html in a browser or serve the folder with a simple static server.

Example:

```bash
python -m http.server 8000
```

Then open http://localhost:8000.
